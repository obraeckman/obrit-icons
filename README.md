# Obrit Icons

A free, open-source set of stylish, modern SVG icons in plain/outline and solid formats, perfect for use in web and mobile applications.

## Features

- **200+ Icons**: A comprehensive set of icons for common use cases.
- **Two Styles**: Available in both **Outline** (2px stroke) and **Solid** formats.
- **Optimized**: Clean SVG code, ready for production.
- **Open Source**: Licensed under the MIT License.

## Installation

### Manual

Simply download the repository or copy the SVG files you need from the `outline` or `solid` directories.

```bash
git clone https://github.com/your-username/obrit-icons.git
```

## Usage

Each icon is provided as a standalone SVG file.

### Structure

- `outline/`: Contains icons with a stroked outline (designed for 24x24px grid, 2px stroke).
- `solid/`: Contains icons with a solid fill.
- `icons-list.csv`: The source of truth for all available icons and their status.

### Example (HTML)

```html
<!-- Outline Icon -->
<svg width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round">
  <!-- Path data from outline/home.svg -->
</svg>
```
